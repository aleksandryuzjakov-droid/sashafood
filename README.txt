САША FOOD — ЯНДЕКС ЕДА СТИЛЬ + СТАТУСЫ

1) Открой server.js
2) Вставь токен бота:
   const BOT_TOKEN = "сюда_токен";

3) Открой CMD в папке проекта

4) Напиши:
   npm install
   npm start

5) Открой сайт:
   http://localhost:3000

Как работает:
- Клиент оформляет заказ
- Открывается страница статуса заказа
- Тебе в Telegram приходит заказ
- Ты нажимаешь кнопки: Принять / Готовится / Едет / Доставлен / Отклонить
- На сайте у клиента статус меняется сам

ВАЖНО:
- Заказы хранятся в памяти сервера.
- Если выключить сервер, старые заказы пропадут.


FIX КНОПОЧЕК:
В этой версии убран setInterval polling, который мог создавать Conflict.
Теперь Telegram слушается одним вечным циклом, поэтому кнопки должны работать стабильнее.
